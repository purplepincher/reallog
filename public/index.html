reallog.ai

A camera sees; a log remembers where.

reallog-agent turns camera events into persistent spatial memory in PLATO, so you can later ask “who came to the front door?” and get an answer grounded in what the camera actually saw.

What This Is

Most camera systems are alert machines. They detect motion, send a notification, and then the moment is gone. If you want to know “who came to the front door last Tuesday?” you are left scrubbing video or trusting a fuzzy memory. The recording exists, but the answer does not.

reallog-agent is a Real-World Scene Logger. Camera frames are turned into short text descriptions, and those descriptions are written to PLATO as tiles. The result is a queryable memory of the physical world: not raw video, but a structured log of what the camera saw, where, and when.

Later, an agent can read those tiles and answer questions in plain language. The README also describes “don the shell”—the org's hermit-crab metaphor for one agent session taking over another's viewpoint—so you can ask the system to look from a different camera's perspective.

Quickstart

Install the package.

pip install reallog-agent
from reallog_agent import RealLogAgent

agent = RealLogAgent(camera_id="security_cam_1")
agent.log_scene("front_door", "person approaching at 14:32", motion=True)
agent.log_scene("back_yard", "bird detected", motion=False)
print(agent.ask("who came to the front door?"))
